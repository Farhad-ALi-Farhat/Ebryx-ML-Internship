# Logistic Regression - Iris Flower Classification

This project demonstrates a complete machine learning classification workflow using **Logistic Regression** on the famous Iris Flower dataset.

The objective is to classify iris flowers into one of three species based on four flower measurements.

---

## Dataset

The Iris dataset is included with Scikit-learn and contains:

- 150 samples
- 4 numerical features
- 3 flower species

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target Classes

- Setosa
- Versicolor
- Virginica
