# 📂 Machine Learning Projects

This folder contains a collection of machine learning projects completed during my internship.  
It covers fundamental concepts including regression, classification, clustering, and model tuning.

---

## 📁 Projects Included

- **01_Linear_Regression_Boston_Housing**  
  Predict house prices using linear regression.

- **02_Logistic_Regression_Iris**  
  Classify iris flower species using logistic regression.

- **03-Titanic_Classification**  
  Predict survival on the Titanic dataset.

- **04-Wine_Classification**  
  Multi-class classification of wine types.

- **05-Breast_Cancer_Classification**  
  Detect malignant vs benign tumors.

- **06-KMeans_Mall_Clustering**  
  Customer segmentation using K-Means clustering.

- **07_GridSearchCV_Hyperparameter_Tuning**  
  Hyperparameter optimization using GridSearchCV.

- **House-Price-Prediction**  
  End-to-end machine learning pipeline project.
