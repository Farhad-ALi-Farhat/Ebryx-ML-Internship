# Week 02 – Classical Machine Learning & Data Preprocessing

This folder contains notes from **Week 02** of my Machine Learning internship. The focus this week was on classical supervised learning algorithms, model evaluation techniques, and essential data preprocessing methods used before training machine learning models.

## Topics Covered

### Classical Machine Learning Algorithms
- Decision Trees
- k-Nearest Neighbors (kNN)
- Naive Bayes

### Model Evaluation
- Train/Test Split
- Cross-Validation
- K-Fold Cross-Validation
- Stratified Cross-Validation

### Data Preprocessing
- Feature Scaling
- Handling Missing Data
- Encoding Categorical Features
- Nominal vs Ordinal Features

### Feature Engineering
- Feature Selection
- Dimensionality Reduction
- Principal Component Analysis (PCA)
- Feature Selection vs Dimensionality Reduction

## Learning Outcome

By the end of this week, I understood how classical machine learning algorithms work, how to properly evaluate models, and how to preprocess data using techniques such as scaling, imputation, encoding, feature selection, and dimensionality reduction. These concepts form the foundation for building effective and reliable machine learning models.
