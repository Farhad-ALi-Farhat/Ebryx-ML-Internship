# What is Machine Learning?

Machine Learning (ML) is a subset of Artificial Intelligence (AI) that enables computers to learn patterns from data and make predictions without being explicitly programmed.

Instead of writing rules manually, the model learns from examples.

## Types of Machine Learning

### 1. Supervised Learning

Uses **labeled data**.

Input → Output

Examples:

- House Price Prediction
- Spam Detection
- Image Classification
- Deepfake Detection

#### Tasks

- Classification
- Regression

---

### 2. Unsupervised Learning

Uses **unlabeled data**.

The model discovers hidden patterns automatically.

Examples:

- Customer Segmentation
- Clustering
- Recommendation Systems
- Dimensionality Reduction

#### Tasks

- Clustering
- Association Rule Learning
- Dimensionality Reduction

---

## Supervised vs Unsupervised

| Supervised | Unsupervised |
|------------|--------------|
| Labeled Data | Unlabeled Data |
| Predicts Outputs | Finds Patterns |
| Classification & Regression | Clustering |

---

## Key Takeaways

- ML learns from data instead of explicit programming.
- Supervised learning uses labeled data.
- Unsupervised learning finds hidden structures in unlabeled data.
