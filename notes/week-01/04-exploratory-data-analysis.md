# Exploratory Data Analysis (EDA)

EDA is the process of understanding and exploring data before training a model.

## Objectives

- Understand the dataset
- Find missing values
- Detect outliers
- Identify relationships
- Discover patterns

---

## Common EDA Steps

- Inspect dataset
- Summary statistics
- Missing values
- Duplicate detection
- Outlier detection
- Correlation analysis
- Data visualization

---

## Common Visualizations

- Histogram
- Box Plot
- Scatter Plot
- Correlation Heatmap
- Bar Chart

---

## Common Pandas Functions

```python
df.info()
df.describe()
df.head()
df.isnull().sum()
df.duplicated().sum()
df.corr()
```

---

## Key Takeaways

- EDA helps understand data before modeling.
- Good EDA often leads to better preprocessing and model performance.
