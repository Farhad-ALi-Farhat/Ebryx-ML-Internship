# Machine Learning Notes

This directory contains my personal notes and summaries from my **Ebryx Machine Learning Internship**.

The notes are organized **week by week**, covering the concepts, algorithms, and practical knowledge learned throughout the internship. Each week's folder contains a `README.md` with an overview and links to individual topics.

## Weekly Notes

| Week | Topics |
|------|--------|
| **Week 1** | Machine Learning Fundamentals, ML Workflow, Bias-Variance Tradeoff, Exploratory Data Analysis (EDA), Gradient Descent, Overfitting vs Underfitting, Evaluation Metrics |
| **Week 2** | Decision Trees, k-Nearest Neighbors (kNN), Naive Bayes, Train/Test Split, Cross-Validation, Feature Scaling, Handling Missing Data, Encoding Categorical Features, Feature Selection vs Dimensionality Reduction |
| **Week 3** | Unsupervised Learning, K-Means Clustering, Hierarchical Clustering, Principal Component Analysis (PCA), Hyperparameter Tuning, GridSearchCV, RandomizedSearchCV |
